How to drop in a new quiz

Click Load Quiz File and select any .json file shaped like this. Save the file anywhere on your computer — nothing is uploaded anywhere, it's read directly by your browser.

{
  "title": "Quiz Title",
  "description": "Optional one-line subtitle",
  "questions": [
    {
      "question": "Question text?",
      "options": ["Choice A", "Choice B", "Choice C", "Choice D"],
      "correctIndex": 1,
      "explanation": "Why that choice is correct."
    }
  ]
}

correctIndex is zero-based (0 = first option). Ask Claude to generate future quizzes in this exact format and they'll drop straight in.
